# Settings

Status: ready-for-agent
Type: task

## What to build

Users can toggle between light, dark, and system theme from a settings modal.

## Acceptance criteria

- [ ] Settings gear icon in home screen top bar
- [ ] Settings modal with theme toggle (light/dark/system)
- [ ] System theme selected by default
- [ ] Theme preference persists across app restarts
- [ ] Theme applies immediately across all screens

## Blocked by

None - can start immediately (parallel with Categories CRUD).
